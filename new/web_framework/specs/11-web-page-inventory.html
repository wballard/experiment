11. Loadout Management (/game/loadouts)

Primary Functions:

  • Complete ability loadout configuration and management
  • Preset loadout slots for different playstyles and scenarios
  • Equipment set integration with ability combinations
  • Loadout activation and sharing capabilities

Design Document References:

  • ES §221-245: Active ability slot management and cooldown systems
  • CS §48-124: Class-specific abilities and ultimate skills
  • ES §181-220: Elemental ability integration and synergy bonuses
  • EI §55-106: Equipment slot coordination with loadout system

Loadout System Features:

Main Loadout Browser (/game/loadouts):
  • 5 preset loadout tabs: Boss Killer, Mob Clear, PvP Duelist, Support, Empty Slot
  • Complete ability visualization: Ultimate, Active Abilities (4 slots), Aura, Weapon Passive
  • Equipment preview with 9-slot layout matching dashboard system
  • Class passive display with inherited bonuses
  • Loadout activation with visual feedback and confirmation
  • Edit Loadout button linking to comprehensive editor
Loadout Editor (/game/edit-loadout):
  • Drag-and-drop interface for ability assignment and management
  • Left panel: Available abilities organized by category (Ultimate, Active, Aura, Weapon Passive)
  • Right panel: Loadout slots with type validation and visual feedback
  • Filter tabs for ability browsing: All, Class, Fire, Ice elements
  • Include options: Equipment sets, Skill point allocation, Cosmetic appearance
  • Save/Cancel functionality with confirmation dialogs
  • Clear All option for complete loadout reset
  • Back navigation to main loadout browser
Ability Categories:
  • Ultimate Abilities: Class-specific and elemental ultimate skills
  • Active Abilities: 4-slot system for class and elemental active skills
  • Auras: Team-wide passive bonuses with special visual styling
  • Weapon Passives: Equipment-based passive abilities
  • Class Passives: Inherited class bonuses (display-only)

Navigation Integration:

  • Positioned between SKILLS and BUILD PLANNER for logical progression flow
  • Equipment integration ensures loadouts include complete gear sets
  • Visual design maintains Courier design system consistency
  • Cross-page state management for seamless loadout editing workflow

MISSION SYSTEM IMPLEMENTATION DETAILS

Mission Success System (v0.3.4)

Reward Generation Algorithm:

  • Currency Parsing: Dynamically extracts Credits and XP from mission reward strings
  • Item Drop Logic: Uses mission item count ranges (e.g., "2-3" → random 2 or 3 items)
  • Random Selection: Pulls items from complete 23-item database with equal probability
  • Rarity Distribution: Natural distribution based on database composition

Visual Reward Presentation:

  • Animated Modal: 0.5s slide-in animation with professional backdrop blur
  • Success Celebration: Glowing green title with cyan mission name display
  • Performance Statistics: Duration, efficiency percentage, and letter grade rating
  • Interactive Item Cards: Hover tooltips showing complete item statistics

Reward Item Features:

  • Rarity-Based Styling: Border colors matching inventory system (common to primal)
  • New Item Indication: Animated "!" badge on first reward with pulsing effect
  • Tooltip Integration: Full item stat display using existing tooltip system
  • Visual Hierarchy: Large item icons with clear name and type display

Navigation Flow:

  • Mission Launch: Automatic item database loading if needed
  • Success Display: Immediate reward presentation with detailed breakdown
  • Inventory Integration: Direct navigation to inventory with reward collection
  • Seamless Return: Continue button returning to mission selection interface

Future Enhancement Roadmap:

  • Mission Failure States: Red-themed modal for unsuccessful missions
  • Partial Success Variants: Graduated reward scaling based on performance
  • Critical Success Bonuses: Enhanced rewards for exceptional performance
  • Failure Analysis Tools: Detailed breakdown of contributing failure factors
  • Equipment Recommendations: Post-failure suggestions for build improvements
