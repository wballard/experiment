COURIER SPECIFICATIONS

Unified Game Design Documents

This directory contains the unified and organized game design documents for Courier, a sci-fi ARPG/looter shooter. All conflicts between previous documents have been resolved and integrated into a coherent design.

01 - Executive Summary
High-level game overview, core innovations, and design pillars. Includes the Power Budget System concept and game identity.
02 - Player Onboarding
New player experience and web-based tutorial system. Web-first gameplay design and browser optimization.
03 - Character Systems
Complete character progression system including level progression, 6 primary attributes, and 5 character classes.
04 - Elemental Combat
Five-element system with status effects, element wheel, and adjacency rules.
04 - Elemental Skills
Comprehensive elemental skill trees with tier-based progression system and combination mastery.
Elemental System
Primary elemental mechanics and Ice tree design with damage modifiers and freeze mechanics.
05 - Equipment & Itemization
Power Budget system and equipment structure with rarity efficiency and strategic choices.
06 - Weapon Systems
Weapon archetypes and modification system with 5 base archetypes and variant creation.
07 - Mission Content
Mission categories and affix system with 3 mission types and weekly rotating modifiers.
08 - Progression & Economy
Character advancement and faction systems with currency and long-term progression goals.
09 - Technical Implementation
Core formulas and calculation methods for power budget validation and system integration.
10 - Implementation Plan
Development roadmap and implementation strategies for the game systems.
11 - Web Page Inventory
Complete inventory of required web pages and their functions with design document references.
12 - Class Skills System
Complete class-specific skill trees with Outlaw class implementation and tier progression mechanics.
13 - Skill Progression Rules
Detailed rules for skill tree progression, tier-gating, prerequisites, and visual connection systems.
14 - Loadouts & Skill Assignment
Build management system for organizing and deploying skill configurations with predefined templates.

Key Design Decisions Resolved

Character Progression

Level cap 60 with Paragon levels, 6 attributes, 60 skill points total

Power Budget

Formula-based equipment limitation system with rarity efficiency multipliers

Elemental System

5 elements with adjacent-only dual selection, percentage-based status buildup

Mission System

3 categories with weekly rotating affixes, removed Raid category
